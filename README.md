# 🤖 AI Arcade: Single-File Masterpieces

A curated collection of fully functional games contained within **single HTML files**, each designed and coded entirely by different Artificial Intelligence models.

This repository serves as a gallery and a benchmark for the creative coding capabilities of Large Language Models (LLMs). No external assets, no complex build steps—just raw HTML, CSS, and JavaScript.

## 🚀 How to Play

Since every game is a self-contained environment, getting started is simple:

1. **download the files** do this by navigating to the releases tab and clicking on the game title under files
2. **open the file** enjoy!

## 🛠️ The Constraints

every game follows these strict rules:

*   **Single File:** All logic (JS), styling (CSS), and structure (HTML) must reside in one file.
*   **No External Assets:** No hosted images or libraries. Everything is rendered via Canvas API, SVG, Base64, or CSS.
*   **(mostly)AI-Authored:** AI makes the file, completely (minor tweaks are allowed)
