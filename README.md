# 🤖 AI Arcade: Single-File Masterpieces

A curated collection of fully functional games contained within **single HTML files**, each designed and coded entirely by different Artificial Intelligence models.

This repository serves as a gallery and a benchmark for the creative coding capabilities of Large Language Models (LLMs). No external assets, no complex build steps—just raw HTML, CSS, and JavaScript.

## 🚀 How to Play

Since every game is a self-contained environment, getting started is simple:

1. **Clone the repo:** `git clone https://github.com`
2. **Browse:** Navigate to any game file in the directory.
3. **Launch:** Double-click the `.html` file to play instantly in your preferred web browser.

## 🛠️ The Constraints

To showcase the true "intelligence" of the models, every game follows these strict rules:

*   **Single File:** All logic (JS), styling (CSS), and structure (HTML) must reside in one file.
*   **No External Assets:** No hosted images or libraries. Everything is rendered via Canvas API, SVG, or CSS.
*   **AI-Authored:** The code is generated by AI, preserving the original logic and "quirks" of the model used.

## 🧠 Why This Project?

As LLMs improve, so does their ability to handle complex game states and procedural generation. This repo acts as a time capsule, documenting how different models approach game design, physics, and UI with limited resources.
